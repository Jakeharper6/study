### 预习内容

**1. cd**

| 符号 | 含义           |
| ---- | -------------- |
| .    | 当前目录       |
| ..   | 上一级目录     |
| -    | 之前的工作目录 |
| ~    | home目录       |

**2. ls**

| 命令  | 作用                                             |
| ----- | ------------------------------------------------ |
| ls -a | all,查看所有文件和文件夹（包含隐藏文件和文件夹） |
| ls -l | long,查看详细属性                                |
| ls -h | humanity,友好方式显示                            |

**3. 文件操作**

| 命令  | 作用                 |
| ----- | -------------------- |
| touch | 新创建一个文件       |
| rm    | (remove)删除一个文件 |
| cp    | (copy)复制一个文件   |
| mv    | (move)移动一个文件   |

**4. 目录操作**

| 命令  | 作用                   |
| ----- | ---------------------- |
| mkdir | 新创建一个文件夹       |
| rm -r | (remove)删除一个文件夹 |
| cp -r | (copy)复制一个文件夹   |
| mv    | (move)移动一个文件夹   |
